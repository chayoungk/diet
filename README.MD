# 주요 기능

 1. 다이어리형 식단 캘린더
* 월간 달력 뷰에서 매일의 섭취 칼로리, 단식 여부(🌙), 운동 여부(🏃‍♂️)를 한눈에 파악.
* 날짜를 클릭하면 해당 일의 식단(아침/점심/저녁/간식), 영양소 달성률, 체중, 운동 시간을 상세히 기록 및 조회.
* 사용자가 직접 원하는 날짜를 '단식일'로 지정하고 해제 가능.

 2. 코칭 및 인사이트
* 오늘의 추천: 전날의 식단 기록과 최근 체중 추세를 분석하여 오늘 필요한 조치를 대시보드에 브리핑.
* 내 몸에 영향을 주는 음식: 섭취 다음 날 체중이 오르거나 내린 데이터를 모아 나의 '위험 음식 TOP 10'을 자동 분석.
* 고위험 조합 패턴: 같이 먹었을 때 유독 체중 증가가 컸던 음식 조합 파악.

 3. 상관관계 차트
* 체중 vs 일일 칼로리: 칼로리 초과가 체중에 미치는 영향을 시각화.
* 체중 vs 영양소: 특정 영양소가 높을 때 체중이 올랐는지 패턴 추적.
* 운동/단식 마커: 운동(🏃‍♂️)과 단식(🌙) 아이콘을 매핑하여, 어떤 활동을 한 날에 체중이 빠졌는지 직관적인 인과관계 확인.
